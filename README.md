# la-hacks-2026

offline survial app

This is a code bundle for Design Trail Safe. The original project is available at https://www.figma.com/design/oYPciBgpS6xgevPqsjOPsZ/Design-Trail-Safe.

## Running the code

Run `npm i` to install the dependencies.
Run `npm run dev` to start Expo.

Useful Expo commands:

- `npm run ios` starts the app in the iOS simulator.
- `npm run android` starts the app in an Android emulator.
- `npm run web` starts the Expo web build.
